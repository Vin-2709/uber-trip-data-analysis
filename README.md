## 1. Project Overview
This project performs **Exploratory Data Analysis (EDA)** on Uber ride data to uncover travel patterns, demand trends, and ride behavior.  
The analysis focuses on trip timing, purpose, and distance to generate meaningful insights.

---

## 2. Objectives
- Understand ride demand patterns  
- Analyze trip purposes and categories  
- Study time-based ride behavior  
- Detect outliers in trip distances  
- Generate insights using data visualization  

---

## 3. Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 4. Dataset
The dataset contains **1,100+ Uber trips** with features like:

- Start Date & Time  
- End Date & Time  
- Category (Business/Personal)  
- Start Location  
- Stop Location  
- Miles traveled  
- Purpose of trip  

---

## 5. Data Preprocessing
- Handled missing values in PURPOSE column  
- Converted date columns to datetime format  
- Extracted time, day, and month features  
- Categorized rides into Morning/Afternoon/Evening/Night  
- Removed null records for clean analysis
